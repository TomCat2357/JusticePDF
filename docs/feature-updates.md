単純なバグ修正・エラーハンドリング・リファクタリング、および最終的に最新版で
採用されなかった仕様は除外しています（末尾の「除外した項目」を参照）。

## 除外した項目

以下は本表から除外しています。

- **単純なバグ修正・エラーハンドリング**: プレビュー順の反転修正、ページ回転時のフリーテキスト表示修正、注釈変更でファイルサイズが増大する不具合、エディタ内 Delete キーの挙動修正、ゴミ箱移動失敗時のエラーハンドリング、PDF 書き込み権限エラー対応 など
- **リファクタリング・内部改善**: 全般的なリファクタリング、`MainWindow` 取得の簡素化、Pixmap キャッシュの evict 対応・重複カード検出の内部改善 など
- **最新版で採用されなかった仕様**:
  - コピー連番のサフィックス `_1`, `_2` 形式（最終的に `(1)`, `(2)` 形式へ置き換え）
  - Python 3.11 ベースのインストーラー（現行は Python 3.14 ベース）
