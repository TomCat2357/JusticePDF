# D&Dについて
基本はエクスプローラーの操作に準拠
D&Dで移動する。ctrlを押しながらならコピー
ファイル又はページの間や一番最初にドロップしたらそこに挿入される。メイン画面限定で、ファイルの上にドロップしたらそこに重ね合わされる
ドロップして重ね合わせ先のファイルや、挿入されたページやファイルについては、選択状態とする。なぜなら何が移動したのかわかりやすくするため。
D&Dはメイン画面からメイン画面でも個別画面から同じ個別画面にも、また個別画面から異なる個別画面にもできる。
D&Dでメイン画面では選択したファイル全体を移動等できる。個別画面では選択したページを移動等できる。
D&Dでメイン画面で複数のファイルを選択して、違うファイルに重ね合わせたら、複数のファイルについて違うファイルに重ね合わされる。このとき、順番としては、前にあるものが上にくる
D&Dでページの間にドロップできる位置ならば、ファイルとファイルの間（又はページとページの間）が判るように何らかの表示を出す。また、メイン画面限定だが、ファイルの上にドロップできる位置ならば、そのファイルが判るように何らかの表示を出す。

# メインウインドウについて
PDFをファイル単位でグリッドに並べる。PDF間にemptyの様な空きは存在しない。させない。
メイン画面では、PDFファイル単位で操作を行う。これは後述の個別PDF画面でいえば全ページを一気に操作していると同義である。
ばらけさせるボタンを押したら、選択ファイルはばらける。ばらけたファイルはとりあえず選択状態になる。

# 個別PDFウィンドウについて
個別ウインドウから全ページを移動した場合は、そのファイルはなくなるし、その個別ウィンドウは閉じられる。
ページをD&Dで移動させた場合、移動もとのページについてリナンバーはしないで一時的に欠番にする。ウィンドウを閉じたらリナンバーする。すなわち1,2,3,4,5で2,3を別個別画面等に移動したら1,4,5となる。また、同一個別画面で、一番最後に移動したら1,4,5,2,3となる。ウィンドウを閉じてまた開けば1,2,3,4,5になる。

# undo,redoについて
個別、メイン関わらず共通で操作を記憶し、元に戻せる。
