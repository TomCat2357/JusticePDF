# PDFas 設計ドキュメント

## 概要

PDFを視覚的に結合・分離できるデスクトップアプリ。`$HOME/Documents/PDFas/` フォルダと同期し、カード形式でPDFを表示・操作する。

## 技術スタック

- Python 3.x
- PyQt6（GUI）
- PyMuPDF / fitz（PDF処理・プレビュー生成）

## アーキテクチャ

```
PDFas/
├── main.py                 # エントリーポイント
├── models/
│   ├── pdf_document.py     # PDFファイルのモデル
│   └── undo_manager.py     # Undo/Redo管理（全体共通）
├── views/
│   ├── main_window.py      # メインウィンドウ
│   ├── page_edit_window.py # ページ編集ウィンドウ
│   └── pdf_card.py         # PDFカードウィジェット
├── controllers/
│   ├── file_controller.py  # ファイル操作（結合・分離・削除）
│   └── folder_watcher.py   # フォルダ監視
└── utils/
    └── pdf_utils.py        # PDF操作ユーティリティ
```

## 作業フォルダ

- パス: `$HOME/Documents/PDFas/`
- アプリ起動時に存在しなければ作成
- このフォルダ内のPDFをカードとして表示
- フォルダ監視で外部変更を自動反映
- 対象: `.pdf` 拡張子のみ（大文字小文字問わず）、直下のファイルのみ

## メインウィンドウ

### レイアウト

```
┌─────────────────────────────────────────────────────────┐
│ [戻る][進む][削除][名前変更][新規空PDF][インポート]      │
│ [回転][全選択][バラバラ][名前順▼][編集日順▼]           │
├─────────────────────────────────────────────────────────┤
│                                                         │
│  ┌─────┐  ┌─────┐  ┌─────┐  ┌─────┐                    │
│  │     │  │     │  │     │  │     │                    │
│  │ PDF │  │ PDF │  │ PDF │  │ PDF │   ← グリッド配置   │
│  │     │  │     │  │     │  │     │                    │
│  ├─────┤  ├─────┤  ├─────┤  ├─────┤                    │
│  │3頁  │  │12頁 │  │1頁  │  │0頁  │   ← ページ数      │

│                                                         │

```

### カードの構成

- 上部: 先頭ページのサムネイル（0ページの場合は空白アイコン）
- 下部: ページ数とファイル名

### マウス操作

| 操作 | 動作 |
|------|------|
| クリック | 選択（他の選択は解除） |
| Ctrl+クリック | 複数選択に追加/解除 |
| Shift+クリック | 最初の選択から範囲選択 |
| 空白エリアをドラッグ | 矩形選択（ラバーバンド） |
| ダブルクリック | ページ編集ウィンドウを開く |
| カードをドラッグ | 移動（グリッド内で位置変更） |
| カードに重ねてドロップ | 結合（ドロップ先の名前を維持、ドラッグ元は削除） |
| Ctrl+ドラッグ | コピー |
| 外部からD&D | フォルダにコピーして追加 |

### キーボードショートカット

| キー | 動作 |
|------|------|
| Ctrl+Z | 戻る |
| Ctrl+Y | 進む |
| Delete | 削除（ゴミ箱へ） |
| F2 | 名前変更 |
| Ctrl+A | 全選択 |

### ツールバーボタン

- 戻る (Ctrl+Z)
- 進む (Ctrl+Y)
- 削除 (Delete)
- 名前変更 (F2)
- 新規空PDF
- インポート
- 回転
- 全選択 (Ctrl+A)
- バラバラにする
- 名前で並び替え（トグル: 昇順/降順）
- 最終編集日で並び替え（トグル: 新しい順/古い順）

## ファイル操作

### 結合

- カードをカードに重ねてドロップ
- ドロップ先のファイル名を維持
- ドラッグ元のファイルは削除

### 分離

- ページ編集ウィンドウからメインウィンドウへページをD&D
- ファイル名: `元ファイル名_1.pdf`, `元ファイル名_2.pdf`, `元ファイル名_3.pdf`...（連番）

### 削除

- Deleteキーまたはツールバーボタン
- Windowsのゴミ箱に移動

### 名前変更

- F2キーまたはツールバーボタン

### 回転

- 選択したカード/ページを時計回りに90度回転
- 複数選択時は全て回転

### バラバラにする

- 選択したPDFを1ページずつ分離
- `元ファイル名_1.pdf`, `_2.pdf`, `_3.pdf`... として作成
- 元ファイルは削除

### 新規空PDF

- 0ページのPDFを作成
- ファイル名: `新規_1.pdf`, `新規_2.pdf`...（連番）

## Undo/Redo管理

### 共通スタック

- メインウィンドウとページ編集ウィンドウで共通のUndoスタック
- すべてのファイル操作を記録し、元に戻せる

### 記録する操作

| 操作 | Undo時の動作 |
|------|-------------|
| 結合 | 元の2つのPDFに分離、削除されたファイルを復元 |
| 分離 | 新PDFを削除、元PDFにページを戻す |
| 削除 | ゴミ箱から復元 |
| 名前変更 | 元の名前に戻す |
| ページ順序変更 | 元の順序に戻す |
| ページ移動（ウィンドウ間） | ページを元のPDFに戻す |
| 回転 | 逆回転 |
| インポート | インポートしたファイルを削除 |
| バラバラにする | 元の1つのPDFに結合 |

### 自動保存

- 操作のたびにファイルを即時更新
- Undoスタックは一定数（100操作）まで保持
- アプリ終了時にUndoスタックは破棄（再起動後はUndo不可）

## フォルダ監視と同期

### 検出するイベント

| イベント | 動作 |
|---------|------|
| ファイル追加 | 新しいカードを表示 |
| ファイル削除 | カードを削除、開いていたページ編集ウィンドウを閉じる |
| ファイル名変更 | カードの表示名を更新 |
| ファイル更新 | サムネイルとページ数を再読み込み |

### 競合の処理

- アプリ内操作中に外部変更があった場合、外部変更を優先
- Undoスタックは外部変更でクリアされる（整合性のため）

---

## ウィンドウ管理

- 複数のページ編集ウィンドウを同時に開ける
- 同じPDFのウィンドウは1つまで（既存ウィンドウをアクティブ化）
- ページ編集ウィンドウ間で直接D&D可能（メインウィンドウを介さずにページ移動/コピー）

---

## 並び替え

- 名前順ボタン: クリックで昇順、再クリックで降順
- 最終編集日順ボタン: クリックで新しい順、再クリックで古い順
- 並び替え状態はアプリ終了後も保持（設定ファイルに保存）

---

## エラーハンドリング

| エラー | 対応 |
|--------|------|
| 破損したPDF | カードにエラーアイコン表示、操作不可 |
| ファイルロック中 | エラーメッセージ表示、操作をスキップ |
| ディスク容量不足 | エラーメッセージ表示、操作を中止 |
